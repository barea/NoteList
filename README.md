# NoteList
Android app connected to REST API, for creat todo List and mange it. user can add tasks to do, update and delete them.

## Libraries and tools included:

* Support libraries
* RecyclerViews and CardViews
* RxJava and RxAndroid
* Retrofit2
* Butterknife
